# Gambler — AI 赌博竞赛

8 个 LLM 代理在 30 轮中通过打工、赌博、进修来竞争，最终资产最高者获胜。

## 快速开始

```bash
cd ai-game-platform
python -m games.gambler.arena 30        # 30 轮
python -m games.gambler.arena 30 42     # 30 轮，随机种子 42
python -m games.gambler.arena --resume  # 从上次中断恢复
```

## 核心规则

### 经济系统

| 参数 | 值 |
|------|-----|
| 初始资产 | $50 |
| 打工收入 | $10/天（保证） |
| 食物开销 | $5/天（自动扣除） |
| 打工净收入 | $5/天 |

### 赌博

| 参数 | 值 |
|------|-----|
| 胜率 | 30% |
| 赢 | 资产 × 3.0 |
| 输 | 资产 × 0.3 |
| 期望值 | 1.11x |
| 公平性 | 同轮所有玩家共享同一随机数 |

赌博的期望值只略高于打工（1.11x vs 1.0x），但波动极大。连续赌输可能直接破产。

### 进修（STUDY）

- 支付学费，锁定 3 轮无收入，完成后工资永久翻倍（可叠加）
- 学费动态：**$45 × 当前工资倍率**
  - 第 1 次：$45（总成本 ~$95，约 10 轮回本）
  - 第 2 次：$90（总成本 ~$170，约 9 轮回本）
  - 第 3 次：$180（总成本 ~$320，约 8 轮回本）
- ⚠️ 进修期间挨饿 → 进修取消，进度清零
- 进修期间锁定：不能做任何决策，无收入，只扣食物

### 助学贷款

进修时如果付不起学费，**自动获得低息助学贷款**：

| 参数 | 值 |
|------|-----|
| 利率 | 5% |
| 还款期 | 15 轮 |
| 宽限期 | 进修期间不还款（毕业后开始还） |
| 覆盖范围 | 学费缺口（需自留 $20 伙食费） |

**生存底线**：即使有助学贷款，你仍需至少 **$15** 付进修期间的伙食费。$0 资产进修 = 必死。

### 医疗灾难

| 参数 | 值 |
|------|-----|
| 发生时机 | 第 10-25 轮之间随机一轮 |
| 发生次数 | **仅一次** |
| 医疗费 | $100/人（所有玩家同时） |
| 付得起 | 直接从资产扣除 |
| 付不起 | 资产归零，缺口转为贷款 |

### 灾难贷款

| 参数 | 值 |
|------|-----|
| 利率 | 20% |
| 还款期 | 20 轮 |
| 示例 | $30 资产 → 缺口 $70 → 贷款 $84 → 每轮还 $4.20 |

**存活线**：灾难前至少攒 $20-30，否则贷款还款会压垮你。

### 饥饿

- 付不起 $5 食物费 = 挨饿 1 天
- **连续 3 天挨饿 = 饿死（淘汰）**
- 进修期间挨饿 → 进修取消 + 挨饿计数

## 每轮结算顺序

```
1. 贷款还款（医疗贷款 + 助学贷款）自动扣除
2. 医疗灾难（如果本轮触发）对所有玩家生效
3. 进修进度推进（如果在进修）
4. 玩家决策：WORK / GAMBLE / STUDY / STUDY_LOAN（先获得收入）
5. 食物费 $5 扣除（然后检查饥饿/破产）
```

## 模型与玩法

每局 8 个不同 LLM 模型，各有不同 persona：

| 默认角色 | 风格 |
|----------|------|
| Alice | 逻辑分析型 |
| Bob | 直觉型 |
| Charlie | 激进主动型 |
| Diana | 谨慎观察型 |
| Eve | 外交共识型 |
| Frank | 沉默洞察型 |
| Grace | 魅力说服型 |
| Henry | 怀疑一切型 |

## 日志与回放

- 摘要：`logs/gambler/<session>/summary.json`
- 直播看板：游戏启动时自动打开浏览器

## 设计理念

进修系统是本游戏的核心策略维度。平衡要点：

1. **前期**：凑不够学费 → 需打工攒钱
2. **中期**：灾难打断风险 → 需把握时机
3. **后期**：回本周期不够 → 需尽早规划

助学贷款降低了进修门槛（不需要攒满 $65，$20 即可），但仍保留了挨饿风险作为硬约束。
